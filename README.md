# 📊 Interactive Sales Dashboard

## 📌 Project Overview

The **Interactive Sales Dashboard** is a data visualization project developed using **Python**, **Pandas**, **Seaborn**, **Matplotlib**, and **Plotly**. The dashboard provides valuable business insights by analyzing sales trends, customer segmentation, product performance, and regional sales through interactive and statistical visualizations.

This project demonstrates the complete data analysis workflow, including data loading, cleaning, exploration, visualization, and reporting.

---

# 🎯 Objectives

- Load and clean sales data
- Perform customer and sales analysis
- Create statistical visualizations using Seaborn
- Build interactive charts using Plotly
- Generate a professional business dashboard
- Provide meaningful business insights and recommendations

# 📉 Visualizations

This project generates the following visualizations:

### 📦 Box Plot
Shows sales distribution across different products.

### 🎻 Violin Plot
Displays regional sales distribution and density.

### 🔥 Correlation Heatmap
Shows relationships between Quantity, Price, and Total Sales.

### 📈 Monthly Sales Trend
Interactive line chart showing monthly revenue.

### 📊 Product Performance
Interactive bar chart comparing product sales.

### 🥧 Regional Sales Distribution
Interactive donut chart representing sales contribution by region.

### 👥 Customer Segmentation
Revenue generated by different customer segments.

### ⭐ Top Customers
Top 10 customers based on total purchase value.

### 🎬 Animated Product Sales
Interactive animation showing product sales over time.
